## Part1

  - Django 설치 및 프로젝트 생성
    - pip install django
    - django-admin startproject mysite
  
  - 개발 서버
    - python manage.py runserver
    
  - polls 앱 시작
    - python manage.py startapp polls
    
  - 첫 번 째 뷰 작성
    - view.py 작성
    - polls/urls.py 작성
    - mysite/urls.py 작성
